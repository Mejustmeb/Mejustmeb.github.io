## How to View the Website

This is a GitHub Pages repository. Once the code is merged to the main branch, the website will be automatically published and accessible at:

**https://mejustb.github.io/**

### Local Preview

To preview the website locally, you can:

1. Open the `index.html` file directly in your web browser
2. Or use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (npx)
   npx serve
   ```
   Then open `http://localhost:8000` in your browser
